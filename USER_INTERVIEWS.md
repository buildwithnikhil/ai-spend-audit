# User interviews (synthetic realism — qualitative intake)

## Interview A — Series B infra founder (Alex)

**Quote:** “Finance sees one Copilot line item; engineering silently spins Cursor everywhere.”

**Insight:** Shadow adoption beats centralized procurement — audits must ingest **multiple overlapping SKUs**.

**Contradiction:** Claims deep cost discipline yet tolerates duplicate assistants because “velocity tax feels cheaper than meetings.”

## Interview B — Growth-stage CFO (Jordan)

**Quote:** “If savings aren’t tied to contract clauses I can negotiate, I ignore them.”

**Insight:** Narratives must bridge modeled deltas → **renewal levers** (true-ups, ramp credits).

**Contradiction:** Demands conservative estimates while pushing aggressive board guidance — tool must label uncertainty honestly (`<$100/mo` path).

## Interview C — Indie hacker / solo SaaS (Sam)

**Quote:** “I’ll pay for fewer tools if someone proves overlap between ChatGPT and Claude APIs.”

**Insight:** API-heavy users distrust subscription benchmarks — usage branch messaging resonates.

**Contradiction:** Wants automation but refuses SSO/connectors — reinforces **no-login** wedge.

### Product learnings

1. Transparency beats cleverness — publish pricing URLs adjacent to recommendations.
2. Shareability drives founder loops — OG cards matter as much as landing polish.
3. CFO persona requires exportability — PDF/print flows remain high priority.
